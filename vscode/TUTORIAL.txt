
VIM PLUGIN SPECIFIC COMMANDS
============================

1. GENERAL
----------

    :w                 Save file
    :q                 Close file
    :q!                Quit without saving
    :qa                Close all files
    u                  Undo last change
    Ctrl+R             Redo last undone change
    .                  Repeat last command
    yy                 Yank line
    p                  Paste
    <leader>s          Save file (custom)
    <leader>w          Close file (custom)

2. SPLITS
----------

    :vsp                Vertical split
    :sp                 Horizontal split
    Ctrl+W+h/j/k/l      Move between splits
    Ctrl+W+H/J/K/L      Move split to left/bottom/top/right
    Ctrl+W+=            Equalize splits

3. TABS
--------

    :tabnew             New tab
    gt / gT             Next / previous tab
    :tabclose           Close tab
    :tabonly            Close other tabs

4. SEARCH
----------

    /pattern            Search forward
    ?pattern            Search backward
    n / N               Repeat search
    * / #               Search current word
    g* / g#             Partial match search
    f<char>             Find next char in line
    F<char>             Find previous char in line
    t<char>             Move until before next char
    T<char>             Move until before previous char

5. VISUAL MODE
---------------

    v                   Visual select (charwise)
    V                   Visual select (linewise)
    Ctrl+V              Visual block
    o                   Jump to other end of selection
    gv                  Reselect last visual selection
    ; / ,               Repeat last f/F/t/T

===============================================================================
                               VS CODE COMMANDS
===============================================================================

1. GENERAL
----------

    Ctrl+S              Save file                 (Vim :w analog)
    Ctrl+W              Close editor              (Vim :q analog)
    Shift+Ctrl+W        Close all editors         (Vim :qa analog)
    Ctrl+Z              Undo last change          (Vim u analog)
    Shift+Ctrl+Z        Redo                      (Vim Ctrl+R analog)
    Ctrl+C              Copy                       (Vim yy/p analog)
    Ctrl+V              Paste                      (Vim p analog)
    Ctrl+X              Cut                        (Vim d analog)
    Space+S             Save file (leader key)    (Vim <leader>s)
    Space+W             Close file (leader key)   (Vim <leader>w)
    Space+R             Reload window             (Vim :source analog)

2. SPLITS / PANES
------------------

    Ctrl+Backslash       Split editor vertically    (Vim :vsp analog)
    Shift+Ctrl+Backslash Split editor horizontally  (Vim :sp analog)
    Ctrl+Alt+Left/Right  Move editor between groups (Vim Ctrl+W+h/l analog)
    Ctrl+Alt+Up/Down     Resize editor group       (Vim Ctrl+W+/- analog)
    Space+=              Equalize all splits       (Vim Ctrl+W+= analog)

3. TABS
---------

    Ctrl+T               Show all tabs / Go to file (Vim :tabnew analog)
    Shift+Ctrl+]         Next tab                  (Vim gt analog)
    Shift+Ctrl+[         Previous tab              (Vim gT analog)
    Ctrl+K Ctrl+W        Close tab                 (Vim :tabclose analog)
    Space+F              Close other tabs          (Vim :tabonly analog)

4. SEARCH
----------

    Ctrl+F               Find in file              (Vim / analog)
    Ctrl+H               Replace in file           (Vim :s analog)
    Shift+Ctrl+F         Search across files       (Vim :BLines analog)
    Ctrl+P               Quick open / Go to file  (Vim :History analog)
    Ctrl+Shift+-         Jump backward in navigation history (Vim Ctrl+O analog)
    Ctrl+Shift+"         Jump forward in navigation history  (Vim Ctrl+I analog)
    Space+/              Open search history      (Vim <leader>/ analog)

5. VISUAL SELECTION
-------------------

    Shift+Arrow          Extend selection          (Vim v analog)
    Ctrl+L               Select line               (Vim V analog)
    Shift+Ctrl+L         Select all occurrences    (Vim visual + gn analog)
    Shift+Alt+Arrow      Column / block selection  (Vim Ctrl+V analog)
    Space+Y              Copy to system clipboard  (Vim <leader>y analog)
    Ctrl+X               Cut                        (Vim d analog)
    Ctrl+V               Paste                      (Vim p analog)

6. CODE / LSP
--------------

    F12                  Go to definition          (Vim gd analog)
    Alt+F12              Peek definition           (Vim gy analog)
    Shift+F12            Show references           (Vim gr analog)
    Ctrl+.                Quick fix / code action   (Vim :CocList analog)
    Ctrl+K Ctrl+I        Show hover documentation  (Vim K analog)
    Shift+Ctrl+R         Rename symbol             (Vim :CocRename analog)
    Alt+Space            Trigger IntelliSense      (Vim <C-space> analog)

7. DEBUG / TERMINAL
-------------------

    Ctrl+`               Toggle integrated terminal
    F5                   Start / continue debugging
    Shift+F5             Stop debugging
    F9                   Toggle breakpoint

8. NOTES
---------

    - Leader key shortcuts require VSCodeVim extension.
    - All Vim motions can be emulated with VSCodeVim, otherwise use native
      VS Code keybindings.
    - Navigation history shortcuts: Ctrl+Shift+- (back), Ctrl+Shift+" (forward)
    - Zen mode can be toggled with Space+Z
    - Quick Open (Ctrl+P) and Command Palette (Shift+Ctrl+P) are main navigation tools.
